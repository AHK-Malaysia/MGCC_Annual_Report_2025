# MGCC Annual Report 2025

Annual Report 2025 of the Malaysian-German Chamber of Commerce and Industry (AHK Malaysia / MGCC) — *A Year of Transition, Growth & Partnership*.

This repository hosts the report as a single self-contained HTML file (`index.html`) with all assets (fonts, images, charts) embedded.

## Viewing locally

Open `index.html` directly in any modern browser, or serve it:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## GitHub Pages

To publish a live version: **Settings → Pages → Source: `main` branch / root**.

> **Note:** GitHub Pages requires a *public* repository on the free tier. Private-repo Pages needs GitHub Pro/Team/Enterprise.

## Notes

- The report uses a horizontal-scroll layout; navigate with on-screen controls or arrow keys.
- Charts are rendered via Chart.js (loaded from CDN).
